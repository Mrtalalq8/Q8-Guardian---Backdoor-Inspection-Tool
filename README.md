# ⚠️ Important Notice

Q8 Guardian is an inspection-only tool.

✔ Performs system analysis  
✔ Displays risk evaluation results  
✔ Identifies suspicious processes and persistence paths  

❌ Does NOT delete files  
❌ Does NOT terminate processes  
❌ Does NOT modify the system  
❌ Does NOT perform automatic remediation  

All decisions remain fully under user control.
